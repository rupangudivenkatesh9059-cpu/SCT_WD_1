# Brew Haven - Responsive Landing Page

## Project Overview
Brew Haven is a simple and responsive coffee shop landing page created as part of the SkillCraft Technology Web Development Internship (Task 1). The project demonstrates a modern landing page with an interactive navigation menu using HTML, CSS, and JavaScript.

## Features
- Responsive landing page
- Fixed navigation bar
- Navigation menu changes color on scroll
- Hover effects on navigation links
- Smooth scrolling
- Hero section
- About section
- Contact section
- Footer

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure


## How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` using Live Server or any web browser.
